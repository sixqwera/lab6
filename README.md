# Чарушников Никита Бас-2 Лабораторная №6

# Задание 1

## Задача 1

### Текст задачи

Разработать класс с тремя целочисленными полями. Создать конструктор копирования.
Разработать метод, вычисляющий максимальную из последних цифр полей. Перегрузить метод
ToString() для формирования строки из полей класса. Реализовать дочерний класс (его
содержание предложить самостоятельно и описать в решении: какой содержательный смысл
имеют поля; реализовать конструкторы; предложить и реализовать 2-3 метода). Протестировать
все конструкторы и другие методы базового и дочернего классов.

### Алгоритм решения

-Создается базовый класс Lab4 с тремя целочисленными полями: _first, _second, _third.

-В классе Lab4 реализуется конструктор с тремя параметрами для инициализации полей.
-Реализуется конструктор копирования, который принимает объект Lab4 и копирует значения его полей в новый объект.
-Разрабатывается метод LastMaxValue, который:
-Берет последнюю цифру каждого поля (через остаток от деления на 10)
-Сравнивает три полученных цифры между собой
-Находит максимальную
-Выводит результат на экран
-Переопределяется метод ToString, который возвращает строку со значениями всех трех полей.
-Создается дочерний класс NewLab, наследующий от Lab4. В него добавляется четвертое поле _fourth, которое хранит дополнительное целое число.
-В дочернем классе реализуется конструктор с четырьмя параметрами, вызывающий конструктор базового класса.
-В дочернем классе создаются два метода:
-Sum — вычисляет сумму всех четырех чисел
-Mul — вычисляет произведение всех четырех чисел
-В методе Main тестируются все конструкторы и методы:
-Создается объект Lab4 через обычный конструктор
-Создается копия объекта через конструктор копирования
-Вызываются методы LastMaxValue и ToString для проверки
-Создается объект NewLab через конструктор дочернего класса
-Вызываются методы Sum и Mul для проверки работы

# Задание 2

## Задача 1

### Текст задачи

LineSegment double x, double y
(координаты начала и конца
отрезка на координатной
прямой)
Определить, попадает ли заданное число
в отрезок. Результат должен быть типа
bool.

### Алгоритм решения

-Создается класс LineSegment, представляющий отрезок на числовой прямой с двумя полями: _x и _y (координаты концов отрезка).
-В классе реализуются конструкторы:
-Конструктор с двумя параметрами для инициализации концов отрезка
-Конструктор копирования для создания копии существующего отрезка
-Перегружаются следующие операторы:
-Оператор ! (унарный) — вычисляет длину отрезка как модуль разности между _x и _y
-Оператор ++ (инкремент) — сдвигает отрезок: левый конец уменьшается на 1, правый увеличивается на 1
-Оператор explicit int — явное преобразование отрезка в целое число (возвращает левый конец)
-Оператор implicit double — неявное преобразование отрезка в число с плавающей точкой (возвращает правый конец)
-Оператор + (бинарный) — сдвигает оба конца отрезка на заданное целое число
-Операторы < и > — проверяют, входит ли целое число в диапазон отрезка (с учетом порядка концов)
-Переопределяется метод ToString для вывода отрезка в формате "[x; y]".
-Создается вспомогательный класс Checker с методами для безопасного ввода чисел:
-CheckDouble(string prompt) — запрашивает число, проверяет корректность ввода
-CheckDouble(string prompt, double min, double max) — запрашивает число с проверкой диапазона
-В методе Main тестируется работа класса:
-Запрашиваются координаты отрезка через Checker
-Создается объект LineSegment
-Выводится длина отрезка через оператор !
-Применяется оператор ++ для сдвига отрезка
-Запрашивается число для проверки вхождения в диапазон
-Через оператор < проверяется, входит ли число в отрезок
-Выводится результат проверки
